# OS Visualizer: Page Replacement Simulator

A high-performance, cinematic web application for simulating Operating System page replacement algorithms. Designed for students and educators to master memory management concepts through real-time visualization and performance analytics.

![OS Visualizer Preview](https://via.placeholder.com/800x400.png?text=OS+Visualizer+Preview)

## 🚀 Features

- **Algorithm Suite**: Implementation of FIFO (First-In First-Out), LRU (Least Recently Used), and Optimal strategies.
- **Real-time Analytics**: Live calculation of Page Faults, Hits, and Hit Ratios.
- **Dynamic Visualization**: Step-by-step animation of memory frames and reference strings.
- **Performance Benchmarking**: Side-by-side comparison of different algorithms using Chart.js.
- **Cinematic UI**: Modern dark-mode aesthetic with fluid transitions and responsive layouts.

## 🛠️ Tech Stack

- **Frontend**: Semantic HTML5, Vanilla CSS3 (Custom Properties), Modern ES6+ JavaScript.
- **Visualization**: Chart.js for performance graphing.
- **Typography**: Inter and Outfit via Google Fonts.

## 📖 How to Use

1. **Select Strategy**: Choose between FIFO, LRU, or Optimal from the dropdown.
2. **Configure Memory**: Set the number of available memory frames (1-7).
3. **Input String**: Provide a comma-separated sequence of page requests.
4. **Simulate**: Click the "Simulate" button to watch the algorithm evolve.
5. **Compare**: Use the "Compare Stats" feature to see how strategies perform against each other for the same input.
